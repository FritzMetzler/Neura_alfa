## Ejecución

Ejecutar generar el compilador `neuratec`:

```zsh
make
```

Para probar
---

```zsh
./neuratec ./ejemplo/return_2.c
```

Correr pruebas
---

```zsh
mix test
```

Limpiar binarios del proyecto
---

```zsh
make clean
```
